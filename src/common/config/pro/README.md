## 项目配置
该目录下的文件主要是用于配置单个项目的信息。  
文件命名规则为：  
aipro-[项目编号]-[项目名称]-conf.json  
e.g  aipro-2016-HN026-guodanian-conf.json  
配置文件的内容如下所示：
```
{ sn: '2016-HN026',
  name: 'guodanian',
  title: '湖南移动-过大年聚合页',
  type: 'web',
  dev: 'pc',
  date: '2016-11-30',
  remUnit: 75,
  compileCss: 'less',
  svn: '' }
```
